# dotfiles
========

## Overview
This is to control and share dotfiles with my several environments.

## Usage
See below.

```shell
./dotfilesLink.sh [os type] [<options>]
```

### os type
`osx`   for mac os  
`win`   for windows os

### options
`--version, -v`   print $FILENAME version  
`--help, -h`      print this
